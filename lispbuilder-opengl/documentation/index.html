Overview

The lispbuilder-opengl package is a part of the Common Lisp Application Builder project which is in turn a part of the umbrella Common Lisp Gardeners project. The goal is to provide OpenGL for Lisp. This packages needs lispbuilder-sdl.

Prerequisites

lispbuilder-sdl and related packages are required prior to the installation of lispbuilder-opengl. The installation of these packages are descibed in the lispbuilder-sdl documentation.

Running the included example

To verify that lispbuilder-opengl is installed correctly, run the gears example by entering:

(asdf:operate 'asdf:load-op :lispbuilder-opengl)
(gl-examples:opengl-gears)

And you should see the animated gears demo. You can drag the mouse to change the viewing angles.

To make the wrappers:

Copy sdlswig.i (the swig interface file) to the opengl include directory
run 'swig -cffi openglswig.i'
